# GeneLab bioinformatics processing pipeline for Mass Spectrometry-based Proteomics Data (TMT-16 Phosphoproteomics Workflow)

> **This page holds an overview and instructions for how GeneLab processes mass spectrometry-based proteomics data using the TMT-16 (Tandem Mass Tag 16-plex) phosphoproteomics workflow. Exact processing commands, GL-DPPD-[TMT16-phospho] version used, and processed data output files for specific datasets are provided in the [Open Science Data Repository (OSDR)](https://osdr.nasa.gov/bio/repo/).**
